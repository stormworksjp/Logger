Stormworks Datalogger System

使い方

データロガーへの記録

  1. 任意の文字列を用意します。(e.g. : player-20200718-test1)
  2. Stormworks LuaブロックでHTTPリクエスト関数を使用し、http://logger.stormworks.jp/record?key=<任意の文字列>&data=<データ>にアクセスします。

データ読み取り

  1. ブラウザでhttp://logger.stormworks.jp/data?key=<任意の文字列>&format=<フォーマット識別子> へアクセスします。
    • 対応フォーマット
      • csv (UNIXタイムスタンプ,データ形式)
      • json

備考

  • このサービスは、ねいろわーくが開発･運用しています。
  • 格納されたデータは、記録されてから1日で取得できなくなります。
  • 予告なくサービスを終了する可能性があります。
  • リポジトリ : stormworksjp/Logger
